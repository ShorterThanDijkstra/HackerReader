1.概述商业技术的角色变化。

·1970年以来，商业技术有哪些不同的名称和角色？

20世纪70年代，商业技术被称为数据处理，其作用是支持现有的业务；20世纪80年代，商业技术的名字变成了信息系统，其角色变成了做生意；20世纪90年代，商业技术变成了信息技术，其角色是改变商业。随着技术在21世纪头10年变得更加先进，它被称为商业智能（或分析），包括数据挖掘、在线分析过程、查询和报告。

·信息技术如何改变商业？

信息技术将时间和地点对于企业的重要性降到最低。不受时间和地点限制的企业可以随时随地提供产品和服务，方便客户。图B-2列举了信息技术改变商业的实例。

·什么是商业智能？

商业智能指的是用于分析组织原始数据并从中获得有益启发的各种软件应用程序。

2.列举商业信息的类型，识别有用信息的特征，并讨论如何存储和分析数据。

·今天的企业可获得哪些类型的信息？

今天的企业可获得的信息包括：（1）业务流程的信息；（2）物质世界的观察；（3）生物数据；（4）公共数据；（5）表示个人偏好或意图的数据。

·你如何应对信息超载？

应对信息超载最重要的是确定你希望实现的四五个关键目标，然后排除与目标无关的信息。

·如何使信息变得有用？

信息管理的有效性取决于四个特征：质量、完整性、及时性和相关性。

·什么是数据分析和大数据？

数据分析是收集、整理、存储和分析大型数据集（“大数据”），以发现对企业目前和未来决策最有用的模式和其他信息。数据分析的一部分是数据挖掘，这是一种寻找隐藏模式和以前未知的数据之间关系的技术。

3.比较互联网、内部网、外联网和虚拟专用网的范围，并解释宽带技术是如何推动Web 2.0和Web 3.0发展的。

·哪些信息技术可以帮助企业管理信息？

信息技术的核心包括互联网、内部网、外联网和虚拟专用网。互联网是一个由成千上万个较小网络组成的巨大网络，任何人只要有计算机和调制解调器就可以使用该网络。内部网是公司内部网络，禁止未经授权的外部人员进入。外联网是一种半私有网络，允许多个企业访问相同的信息。虚拟专用网通过常规互联网线路，创建安全连接或“隧道”的专用数据网络。

·什么是Web 2.0？

Web 2.0是一组工具，允许人们在线建立社交和业务联系，共享信息和协作项目，如博客、维基、社交网络和其他在线社区，以及虚拟世界。Facebook、YouTube和微博网站Twitter都是最大的Web 2.0企业，其中所有内容都是由普通人创建的。

·什么是Web 3.0？

Web 3.0是一种技术组合，可以增加智能，与网络互动。可以将Web 3.0描述为个人网络，而将Web 1.0描述为静态网络，Web 2.0描述为社交网络。Web 3.0技术由三个基本组件构成：语义网、移动网络和沉浸式互联网，它们都连接着许多人所说的物联网。

·什么是物联网？

物联网是指利用传感器、摄像头、软件、数据库和大规模数据中心，将普通物体连接到互联网上的技术。

4.阐释虚拟网络，并讨论云计算的优缺点。

·什么是虚拟网络？

虚拟网络是允许基于软件的联网计算机运行多个操作系统和程序以及共享存储的过程。虚拟网络可以是外部的，也可以是内部的。外部网络（external networking）将网络视为单个资源池，无论其物理组件如何（即联网的计算机共享资源），都可以访问这些资源。内部网络（internal networking）与所有联网的计算机共享一台中央计算机的资源。

·什么是云计算？云计算有哪些优缺点？

云计算是一种虚拟化存储形式，企业的数据和应用程序存储在通过互联网（云）访问的非现场数据中心。云计算的优点包括降低软件成本、提高安全性、灵活可变的容量、降低设备成本、更便捷的访问。缺点是数据的安全性、稳定性和控制性方面的问题。

5.评估受信息技术影响的人力资源、安全性、隐私性和稳定性问题。

·信息技术对企业管理有何影响？

计算机淘汰了一些中层管理职能，从而使组织结构扁平化。员工还可以借助计算机在家工作。从消极的方面来说，计算机有时会让信息落入坏人之手。互联网上可用的海量信息引发了人们对隐私问题的关注。要在获取私人信息的自由和个人维护隐私的权利之间找到平衡，仍有待继续探讨。
